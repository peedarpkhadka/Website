Python Programming (Beginner Friendly)

Variables and Data Types

Variables store data, like containers. Python supports data types like:

  • Integers: age = 25
  • Floats: price = 19.99
  • Strings: name = "Alice"
  • Booleans: is_student = True

Example:


                age = 30
                name = "Bob"
                print(name, "is", age, "years old.")
                # Output: Bob is 30 years old.

Conditional Statements

Use if, elif, else to make decisions.


                score = 85
                if score >= 90:
                    print("Grade: A")
                elif score >= 80:
                    print("Grade: B")
                else:
                    print("Grade: C")
                # Output: Grade: B

Loops

For Loop: Iterates over a sequence (e.g., list, range).


                for i in range(5):
                    print(i)  # Outputs: 0, 1, 2, 3, 4

While Loop: Runs as long as a condition is true.


                count = 0
                while count < 3:
                    print("Hello")
                    count += 1  # Outputs: Hello (3 times)

Functions

Functions are reusable blocks of code.


                def greet(name):
                    return f"Hello, {name}!"
                print(greet("Alice"))  # Output: Hello, Alice!

Lists and Dictionaries

Lists: Ordered, mutable collections.


                fruits = ["apple", "banana", "cherry"]
                fruits.append("orange")
                print(fruits[1])  # Output: banana

Dictionaries: Key-value pairs.


                student = {"name": "John", "age": 20}
                print(student["name"])  # Output: John

File Handling

Read from or write to files.


                # Writing to a file
                with open("example.txt", "w") as file:
                    file.write("Hello, World!")

                # Reading from a file
                with open("example.txt", "r") as file:
                    content = file.read()
                print(content)  # Output: Hello, World!

Why Python?

Python’s simplicity makes it ideal for beginners, yet it’s powerful for advanced tasks like AI and web development.
