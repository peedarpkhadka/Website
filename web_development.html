Web Development Basics

HTML, CSS, JavaScript Basics

  • HTML: Structures content (e.g., headings, paragaraphs, structure). 
    
                        <h1>Welcome>/h1>
                        <p>This is a paragraph.>/p>
  • CSS: Styles content (e.g., colors, layouts) 
    
                        h1 {
                            color: blue;
                            text-algin: center;
                        }
  • JavaScript: Adds interactivity (e.g., event handling, slideshow, button clicks) 
    
                        document.getElementById("myButton").addEventListener("click", () => {
                            alert("Button clicked!");
                        });

Backend with PHP

PHP handles server-side logic, like processing forms or querying databases.


                <?php
                    if ($_SERVER["REQUEST_METHOD"] == "POST") {
                        $name = $_POST["name"];
                        echo "Hello, $name!";
                    }
                    ?>
                    <form method="post">
                        Name: <input type="text" name="name">
                        <input type="submit">
                    </form>
