Inheritance

Classes can inherit properties and methods form others.


                class Puppy(Dog):
                    def play(self):
                        return f"{self.name} is playing!"

                my_puppy = Puppy("Max")
                print(my_puppy.bark())  # Output: Max says Woof!
                print(my_puppy.play()) # Output: Max is playing!

Polymorphism

Objects of different classes can share the same method name with differnt behaviours.


                class Cat:
                    def speak(self):
                        return "Meow"

                class Dog:
                    def speak(self):
                        return "Woof"

                pets = [Cat(), Dog()]
                for pet in pets:
                    print(pet.speak())  # Output: Meow, Woof

Encapsulation

Restrict access to data using private attributes (denoted by __).


                class BankAccount:
                    def __init__(self, balance):
                        self.__balance = balance
                    
                    def deposit(self, amount):
                        self.__balance += amount
                        return self.__balance

                account = BankAccount(1000)
                print(account.deposit(500))  # Output: 1500
                # print(account.__balance)  # Error: Attribute not accessible

Why OOP?

OOP organizes code for scalability, making it easier to manage large projects.
